# Modern Luxury Restaurant Project Guidelines

Always prioritize:
1. **Performance**: Optimize images, use server components where possible, minimize bundle sizes.
2. **GPU-Accelerated Animations**: Use `framer-motion` for fluid, physics-based UI transitions. Prefer layout animations and hardware-accelerated CSS properties (transform, opacity) to prevent layout shifts.
3. **WCAG Accessibility**: Ensure high-contrast ratios for text readability, proper semantic markup, and full ARIA keyboard accessibility.
4. **Minimal, Elegant Design**: Maintain an uncluttered aesthetic featuring sophisticated color palettes, spacious layouts, and beautiful typography.
